about=VectorToMap is a powerful automation tool designed to streamline the creation of high-quality print layouts directly from vector layers.
 Instead of manually configuring complex Atlas settings, this plugin allows users to generate consistent, professional map books in seconds.
 Key Features:
 - Dynamic Page Generation based on features or attributes.
 - "Iron-Lock" Geometry Presets (75%% Height or Square layouts).
 - Smart Scaling Engine: calculates optimal zoom based on the longest dimension to prevent clipping.
 - Advanced Attribute Display: Supports HTML Form and Individual bordered labels.
 - Performance Focused: Manual Render Preview button to handle heavy datasets.
